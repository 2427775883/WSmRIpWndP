# 项目简介
## 核心功能
- 学生端：
  - 注册登录
  - 修改密码
  - 可视化选择自习室与座位
  - 预约时段管理（支持多日期选择）
  - 查看历史预约记录
  - 违规警告通知
- 管理端：
  - 自习室容量管理
  - 学生信息维护
  - 预约记录监控
  - 警告记录管理
  - 系统公告发布
## 适用场景
- 大型图书馆座位预约管理
- 教育机构自习室资源分配
## 技术特点
- 核心框架：Python3.8＋ Django 4.0
- 前端：Bootstrap 5 + Django模板引擎
- 后台管理：django-simpleui
- 数据库：MySQL（默认使用SQLite）
- 会话管理：Django Session
- 文件存储：Django Media模块
